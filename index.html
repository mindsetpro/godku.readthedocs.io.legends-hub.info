Mod Commands

Command Description
!invert Use this command to manually edit the file.
!14stars Make all your characters 14 stars.
!Zenkai Make all zenkai characters zenkai 7.
!Soulboost Max the soulboost of all your characters.
!Artboost Max the art boost to 99 for all characters.
!Costume Give you the alternate color of all characters.
!Equip Choose between absurd equipment power or maxed equipment with official stats (Second option is in work in progress).
!Level Change the level of all characters (Only in friendly battle).
!14,7 Make the commands !14stars and !Zenkai at the same time.

!Allmods Make the commands !14,7 and !Allboost at the same time.
!Ultimate Make the commands !Allmods and !Costume at the same time.
!Visual Give you all the characters in the game (WARNING: Like the command name says, it's only visual, you are not able to play the given characters!).

!Shallot Edit shallot costume, special move, or transformation even if you don't have them.

!Skip_tuto Skip the king kai tutorial on a new account.

Mod Apk

You can download godku mod apk. There are different versions to fit what you want.

What's the difference between 32bit and 64bit?

32bit is more compatible and can sometimes be more fluid while 64bit is more stable and will experience fewer crashes.

What's the difference between basic and injection version?

Injection version maxes all your characters like the bot does but cannot be deactivated. While the basic version doesn't do that so you still have to use the bot to max your characters.
